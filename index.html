1. Les différents formats

JSON

Javascript Object Notation

C'est le plus utilisé.
Il se lit rapidement, il est léger mais très peu descriptif.

XML

Extensible Markup Language

Le plus lourd et le plus descriptif.
Souvent attaché à une documentation de description. Il est plus vieux que JSON et de moins en moins utilisé.

CSV

Comma-separated values

Le plus vieux. Le plus simple et le plus cryptique. Il s'obtient à partir d'un document excel ou d'un autre tableur. Vous pouvez aussi le générer.

Quasiment plus du tout utilisé, il reste un moyen simple d'intégrer des données Excel dans un programme ou une webapp.
